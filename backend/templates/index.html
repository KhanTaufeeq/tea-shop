Template inheritance allows you to build a base “skeleton” template that contains all the common elements of your site and defines blocks that child templates can override. All the block tag does is to tell the template engine that a child template may override those portions of the template. Content within a {% block %} tag in a parent template is always used as a fallback. If you use {% extends %} in a template, it must be the first template tag in that template.
